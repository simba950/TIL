Publisher?

-> Publisher는 Combine 프레임워크에서 제공하는 비동기 데이터 스트림을 생성하는 객체다. -> 특정 값이 변경될 때마다 새로운 값을 방출하는 역할을 함. -> Publisher가 방출하는 데이터를 받아서 처리하는 게 Subscriber다.

protocol Publisher {
    associatedtype Output
    associatedtype Failure: Error

    func receive<S>(subscriber: S) where S : Subscriber, 
                                        Self.Failure == S.Failure, 
                                        Self.Output == S.Input
}
  • Output : 방출하는 값의 타입
  • Failure : 실패 시 반환할 에러 타입 -> 특정 타입의 값을 주기적으로 내보낼 수 있고, 실패할 수도 있는 존재
